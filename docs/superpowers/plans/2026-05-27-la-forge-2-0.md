# 🎨 La Forge 2.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement "La Forge 2.0", extending the universe fusion system into an interactive Mini Visual Novel (VN) with a co-creation "Director Mode".

**Architecture:** A new `VisualNovelService` to handle script generation and image orchestration, a Django API for state management, and a dedicated React frontend for playback and editing.

**Tech Stack:** Django, LLMService (vLLM), DiffusersAdapter (SDXL/Flux), React, Framer Motion, Zustand.
