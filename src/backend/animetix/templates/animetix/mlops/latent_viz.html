🧠 C'est quoi ?

L'Espace Latent est une représentation mathématique où chaque œuvre est un point dans un univers à 768 dimensions. Plus deux points sont proches, plus l'IA considère que les œuvres sont similaires.

📐 Visualisation 3D

Nous utilisons l'algorithme UMAP pour projeter ces 768 dimensions en 3 dimensions visibles, tout en préservant les clusters (regroupements) naturels de données.

🎮 Application

Dans le jeu, quand vous proposez un titre, nous mesurons la distance entre votre choix et le secret dans cet espace pour calculer votre score de proximité.
