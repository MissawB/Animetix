Analyse Vidéo : Stratégies de Conception

Objectif : Métadonnées, Mini-jeu "Devine l'épisode" & Résumés Globaux

A

Hybride Cloud (Gemini 1.5 Flash)

Le plus puissant et précis.

  • Traitement de vidéos jusqu'à 1h via l'API gratuite.
  • Résumé global ultra-précis (contexte narratif).
  • Identification native des moments clés pour le mini-jeu.
Recommandé (SOTA 2026)
B

Full Local (FFmpeg + CLIP/Moondream)

Confidentialité totale, 0 dépendance cloud.

  • Extraction de frames clés via FFmpeg.
  • Vectorisation (CLIP) pour comparer les images.
  • Léger pour les métadonnées visuelles, plus dur pour les résumés narratifs.
Flux Mini-jeu (Option A)
Vidéo -> Gemini -> Points clés -> Quiz
"Dans cet extrait, quel personnage fait son apparition ?"
Flux Mini-jeu (Option B)
Vidéo -> FFmpeg -> Vector DB -> Match
"Cette image correspond à l'épisode 12 (89% match)"
